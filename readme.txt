== Description ==

> International Shipment label generation.
> At the moment, no type of contract with the provider is required to make use of this module.
> No override files are required for proper functionality of the module.

= BACK OFFICE (SHIPPING ): =

[DHL Parcel shipping](https://wordpress.org/plugins/automated-dhl-parcel-shipping-labels) plugin is deeply integrated with [HITShipo](https://app.hitshipo.com/). So the shipping labels will be generated automatically. You can get the shipping label through email or from the order page.

 This plugin also supports the manual shipments option. By using this you can create the shipments directly from the order page. [HITShipo](https://app.hitshipo.com/) will keep track of the orders and update the order state to complete.
